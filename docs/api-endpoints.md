# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `GET /api/users`
- `POST /api/users`
- `PATCH /api/users`

### User follows

- `GET /api/user_follows`
- `POST /api/user_follows`
- `PATCH /api/user_follows`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Playlists

- `GET /api/playlists`
  - Playlists index
- `POST /api/playlists`
- `GET /api/playlists/:id`
- `PATCH /api/playlists/:id`
- `DELETE /api/playlists/:id`

### Playlist follows

- `GET /api/playlist_follows`
- `POST /api/playlist_follows`
- `PATCH /api/playlist_follows`

### Songs

- `GET /api/songs`


### Playlist members

- `GET /api/playlist_members`
- `POST /api/playlist_members`
- `PATCH /api/playlist_members`

### Albums

- `GET /api/albums`

### Artists

- `GET /api/artists`
