# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `GET /api/users`
- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Playlists

- `GET /api/playlists`
  - Playlists index
- `POST /api/playlists`
- `GET /api/playlists/:id`
- `PATCH /api/playlists/:id`
- `DELETE /api/playlists/:id`

### Songs

- `GET /api/songs`

### Albums

- `GET /api/albums`

### Artists

- `GET /api/artists`

### Follows

- `GET /api/follows`
  - get current users follows
